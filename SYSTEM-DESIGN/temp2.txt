=> How to answer System Design interview

    2) Clerify function and non-function requirements
    3) Estimate amount of Data
    4) Design system at high level
    5) decide how server and client server communicate
    6) Deep dive in Design
    7) identify bottle neck and scaling oppertunity
    8) is there single point of failure is replication is neccessary
    9) edge cases
    10) how to scale system 
    11) Review and wrap up
